# jamile.dev

A personal engineering blog built with [Astro](https://astro.build/), featuring bilingual posts (English and Portuguese) about code, architecture, debugging, and technical reflections.

## Tech Stack

- **Framework**: [Astro](https://astro.build/) 5.5.2
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with Typography plugin
- **Language**: TypeScript
- **Features**: 
  - Bilingual support (English / Portuguese)
  - RSS feeds (per language)
  - XML sitemap with i18n support
  - Type-safe content collections
